# Sabor Express 🍽️

Sistema em Python para gerenciamento e avaliação de restaurantes. 
Este projeto foi desenvolvido como parte dos meus estudos em lógica de programação e sistemas de desenvolvimento.

# Funcionalidades
- Cadastrar novos restaurantes.
- Listar todos os restaurantes cadastrados.
- Alternar o estado (ativo/desativado) dos restaurantes.
- Visualizar um ranking dos restaurantes com base em suas notas.

# Tecnologias utilizadas
- Python
- Git/GitHub

# Como executar
1. Clone este repositório:
   `git clone https://github.com/devjosesousa/sistemas-restaurantes.git`
2. Execute o arquivo principal:
   `python app.py`
